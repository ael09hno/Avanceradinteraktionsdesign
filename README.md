Avanceradinteraktionsdesign

Instruktioner.

1. Innan man börjar koda så kör git pull alternativt synka via Windows github
2. Koda ändringar. Spara.
3. git status, ignorera den rödmarkerade filen som heter .metadata/.
4. git add filnamn, detta kommando kan göras flera gånger i rad ifall flera filer har ändrats i.
5. git commit -m 'meddelande'
6. git push
